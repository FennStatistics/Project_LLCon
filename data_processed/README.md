# data_processed

This directory contains outputs derived from the XML corpus. The primary
artifact is the SQLite database created by the preprocessing notebook.

Expected output
- `data_processed/corpus.sqlite`: searchable SQLite database of `<absatz>`
  units, generated by `notebooks/prepare_corpus.ipynb`.

Metadata fields
- Each `<absatz>` row includes `year`, `band`, and `entscheidungsart` derived
  from `data/Wendel_Korpus_BVerfG/Metadaten2.7.1.csv` (keyed by `dateiname`).

Provenance
- Source: `data/Wendel_Korpus_BVerfG/xml`
- Extraction unit: `<absatz>`
- Tooling: `lxml` for parsing; `sqlite3` with FTS5 for indexing.

Notes
- Do not edit outputs in place. Re-run the notebook to regenerate.
- The XML corpus remains the source of truth.
